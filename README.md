# Customer-Demographics-and-Purchase-Behavior-Analysis

This project analyzes customer demographics, product categories, and purchase patterns for XYZ Retail, using an Excel-based end-to-end analysis workflow.
The goal is to derive actionable insights that support marketing, segmentation, and inventory planning.

# Objective

To identify key customer groups, spending trends, and product/category performance using data visualization and exploratory analysis in Excel.

# Dataset Overview

- Rows: 27,397

- Columns: 9

- Fields: Customer ID, Age, Gender, Region, Product Category, Purchase Date, Amount, etc.

# Data Cleaning Performed

- Bold headers & alternate row shading

- Currency formatting for Amount

- Fixed missing/inconsistent entries

- Validated dates

- Created Age Groups (18–25, 26–35, 36–45, etc.)

# Strategic Insights

- Target Age Group: 26–45 age group drives maximum revenue and orders

- Inventory Planning: Stock up for peaks in January, May, October

- Category Strategy: All categories perform similarly → opportunity for bundling & cross-selling

- Regional Marketing: Run custom campaigns for North & West

- Revenue Potential: Consistent spending across age groups
